# MongoDB (mongodb)

MongoDB is a source-available cross-platform document-oriented database program. Classified as a NoSQL database, MongoDB uses JSON-like documents with optional schemas.

**URL:** [https://www.mongodb.com](https://www.mongodb.com)

## APIs

### MongoDB Atlas Administration API

The MongoDB Atlas Administration API allows you to programmatically manage your MongoDB Atlas clusters, projects, organizations, users, and other resources.

- **Human URL:** [https://www.mongodb.com/docs/atlas/api/](https://www.mongodb.com/docs/atlas/api/)
- **Base URL:** `https://cloud.mongodb.com/api/atlas/v2`

### MongoDB Atlas Data API

The Atlas Data API lets you read and write data in MongoDB Atlas with standard HTTPS requests, without the need for a MongoDB driver.

- **Human URL:** [https://www.mongodb.com/docs/atlas/app-services/data-api/](https://www.mongodb.com/docs/atlas/app-services/data-api/)
- **Base URL:** `https://data.mongodb-api.com/app`

#### Tags

- Data Access
- HTTPS
- REST API

#### Properties

- [Documentation](https://www.mongodb.com/docs/atlas/app-services/data-api/)
- [Examples](https://www.mongodb.com/docs/atlas/app-services/data-api/examples/)

### MongoDB Atlas App Services Admin API

Admin API for MongoDB Atlas App Services (formerly Realm), used to manage applications, services, functions, and triggers.

- **Human URL:** [https://www.mongodb.com/docs/atlas/app-services/admin/api/v3/](https://www.mongodb.com/docs/atlas/app-services/admin/api/v3/)
- **Base URL:** `https://realm.mongodb.com/api/admin/v3.0`
